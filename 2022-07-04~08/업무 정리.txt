[목표] 
사용자가 특정한 서식 없이 입력한 데이터들을 정제하고 키워드를 토대로 매핑하는 프로그램을 설계한다.

사용 언어 : JAVA
자료 구조 : 이중 리스트, 해시맵
데이터 셋 : A(400만 건) B(20만 건) C, D
결과물 : A의 키워드와 B의 키워드를 정제한 후, 각 키워드를 토대로 매핑(일치 또는 포함 관계를 따지며)한다. 매핑 결과 output을 csv 파일 형식으로 출력하고,
        이외의 필요한 정보는 C, D에서 추출한다.
로직 : 일치 또는 포함 관계를 판별할 때 최악의 경우로 가정하면 400만 * 20만의 반복문을 수행한다. 
      따라서 B 데이터를 유사도에 따라 페이지를 나눈 후, A의 각 데이터마다 가능성을 체크하고 가능성이 있는 페이지에만 접근하여 반복문을 수행한다. 
      이외의 데이터 접근은 해시맵을 사용하여 탐색 속도를 높인다. 또한, 정제를 모두 진행한 후 매핑을 하는 것보다, 정제한 데이터는 따로 메모리에 올릴 필요가 
      없으므로 정제하고 매핑하고, 또 정제하고 매핑하고를 반복하는 방식으로 로직을 설계하였다.
      
결과물 성능 : 정확도 100% 조건 만족도 100% 속도 20,000/1m = 4,000,000/200m = 4,000,000/3.33333h

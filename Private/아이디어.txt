
정리

1. 알리기 기능

환경 미화원의 손길이 닿지 않아서 또는 불법 투기 쓰레기로 인해서 더러워진 길거리 사진을 찍고, 위치를 알린다.

2. 알림 기능

사용자 위치 정보를 불러와 사용자 주변이 오염되었다는 알림을 주어, 지나가다가 정화작업에 참여할 수 있게끔 유도한다.

3. 커뮤니티 기능

자신의 정화 작업 전 / 후 사진을 올려 사람들과 공유한다. 추천을 가장 많이 받은 사람에게는 추가적이 보상이 주어진다.(표창장, 지역 포인트 등)

4. 쓰레기통 위치 알려주기 기능

쓰레기를 주웠거나, 또는 내 손에 쓰레기가 있을 때, 주변 쓰레기통들의 위치를 보여준다.

5. 정화하기 기능

정화 작업을 모집하는 글이 올라오고, 참여할 수 있다.
-> 개인적으로도 정화 작업이 가능하게끔 하고 싶은데 방법이 생각이 안납니다..
-> 관리자의 통솔 하에 정화 작업이 이루어진다면 너무 비효율적일 것 같고, 실현 가능성도 적어 보입니다.

7. 시각화 기능

길거리가 정화되어 지구가 깨끗해진다는 걸 시각적으로 확인할 수 있게끔 보여준다.

+ 관리자 역할

관리자는 올라온 글들을 보고 오염 정도를 판단하여 개인 정화 작업이 불가능하다면 정화 작업 인원을 모집한다.
개인 정화 작업이 가능하다면? ????

-> 개인적으로 정화 작업이 가능한 곳도 취급을 하면 좋겠습니다.
